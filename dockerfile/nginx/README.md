## `Docker Compose`

```yaml
version: "3"
services:

  nginx:
    image: "khs1994/nginx:1.15.5-alpine"
    ports:
      - "80:80"
      - "443:443"
    environment:
      - TZ=Asia/Shanghai
    volumes:
      - ./app:/app:rw
      - ./conf.d:/etc/nginx/conf.d:ro
```

## `$ docker run`

```bash
$ docker run -dit \
         -e TZ=Asia/Shanghai \
         -p 80:80 \
         -p 443:443 \
         -v $PWD/app:/app \
         -v $PWD/conf.d:/etc/nginx/conf.d \
         khs1994/nginx:1.15.5-alpine
```
