# 快速上手

## 开发环境

```bash
$ ./docker-lnmp.sh devlopment

$ curl 127.0.0.1

Welcome use khs1994-docker/lnmp

```

在 `./app` 目录下开始 PHP 项目开发。

## 生产环境

开启 `容器即服务( Caas )` 之旅。

```bash
$ ./docker-lnmp.sh production
```

尽可能拉取镜像，避免构建镜像占用时间（部署 Docker 镜像私有服务器）。

## 停止

```bash
$ docker-compose stop
```

## 销毁

```bash
$ ./docker-lnmp.sh devlopment-down | production-down
```
