## 配置 `php.ini`

> 务必按以下步骤配置扩展文件夹路径，近来 brew 已将 PHP 从 tap 移到了核心仓库。

```bash
# 获取 pecl 安装扩展地址

$ pecl config-get ext_dir

/usr/local/lib/php/pecl/20170718

# 修改 php.ini

extension_dir = "/usr/local/lib/php/pecl/20170718"
```

> pecl 安装扩展时会自动在 php.ini 文件开头添加扩展
