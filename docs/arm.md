# ARM 架构

[arm32v7 Docker image](https://hub.docker.com/u/arm32v7/) 是树莓派官方系统可以直接使用的。

[arm64v8 Docker image](https://hub.docker.com/u/arm64v8/) 即 arm 64位 架构在第三方系统（如 [Rancher OS](rancher.com/rancher-os)）可以使用，树莓派官方没有发布 arm 64 位系统。

注意 [armhf Docker image](https://hub.docker.com/u/armhf/) 已经废弃，已经转移到了上述两个项目中，更多信息请查看下边的参考链接。
